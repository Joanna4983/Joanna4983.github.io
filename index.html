MetaEx: Image Metadata Extraction Tool

MetaEx is an image metadata extraction tool. It takes in an image and displays the metadata contained within it. There is also a scraping functionality where the tool can scrape a given URL for images and download them.

It's written entirely in Python.

MITM

MITM is a Man In the Middle tool that has been developed as part of my final year project. It aims to educate people about the dangers posed by MITM attacks and demonstrate how these attacks take place and how to avoid them.

It's written entirely in Python & Shell.
